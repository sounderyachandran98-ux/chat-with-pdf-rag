# Chat with Any PDF - RAG Application

A Retrieval-Augmented Generation (RAG) application that allows users to upload PDF documents and ask questions about their contents.

## Features

- Upload any PDF
- Semantic search using FAISS
- HuggingFace Embeddings
- Google Gemini 2.5 Flash
- Page-level source citations
- Interactive Chainlit UI

## Tech Stack

- Python
- LangChain
- FAISS
- HuggingFace Embeddings
- Google Gemini
- Chainlit
- PyPDF

## Run

----bash
chainlit run app_upload.py
----
